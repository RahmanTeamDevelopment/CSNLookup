Manual


CSN Lookup is a simple online tool for validating clinical annotations of variants. It checks whether the inputted annotations follow the CSN (Clinical Sequencing Notation) v1.0 specification and correspond to possible variants in the inputted transcripts. For valid CSN annotations, the tool displays the corresponding genomic coordinates, reference and alternate alleles, the location within the transcript and functional consequence class of the variants. For invalid annotations, the reasons for failing the CSN specification are provided.

How it works

After first assessing the syntax of each inputted annotation, CSN Lookup attempts to convert the annotation to VCF coordinates with regards to the particular transcript and uses CAVA v1.2.0 to re-annotate each variant. In this multi-step process, CSN Lookup is able to detect problems in either the syntax or consistency of annotations (i.e. valid annotations are those that are in CSN v1.0 format and correspond to real sequence variants).

Submitting a query

Queries can be submitted on the Submit query page. Each inputted line must include two columns (either space or tab separated) providing the CSN annotation in question and the ENST ID of the transcript, respectively. A maximum of 500 annotation/transcript pairs can be submitted.

Example query:

c.421A>G_p.Thr141Ala         ENST00000335137
c.490-40_490-37dupGAAA   ENST00000428173

The user should also specify the transcript database to be used by selecting from the available databases in the drop-down list.

The results page

The results of a processed query are presented in a scrollable table on the Results page. The first two columns ('CSN query' and 'Transcript ID') identify the input record, while the third column ('Comment') reports any particular issues with the annotation. Invalid annotations are highligthed in red. Clicking on a valid annotation opens up a window providing more details of the variant such as its genomic coordinates, alleles, location and functional consequence class. Results can also be downloaded as tab-delimited .txt file by clicking on the Download button.

Transcript catalog

The Transcript catalog page provides a search interface allowing to check which genes and transcripts are included in the available transcript databases before submitting a query. One can search either by gene or transcript name.
